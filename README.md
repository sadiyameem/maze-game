# Maze Game

# Description
- This game will test your skills in finding your way out of a maze.

# Features
- Use arrow keys
- Use W A S D keys
- Reload the browser to play again

# How I made this
- HTML
- CSS
- JavaScript

# Motivation
- I made a project just like this when I did GirlsWhoCode and wanted to do it again on vscode

# Challenges/Learned
- A challenge was to make the play again button active but instead I used the reload button on the browser
- Another challenge was to make the square appear on the bar instead of under the bar
- I learned how to use a physics engine (Matter.js) to create a 2D game
